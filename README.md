# Movies App

A Flutter movie browsing application built with **Clean Architecture** and **BLoC** for state management.  
It connects to a **Laravel REST API** backend with a MySQL database.

##  Features

- Browse popular and latest movies  
- Read movie details and synopsis  
- View cast and actor profiles  
- Add or remove movies from favorites  
- Browse movies by genres (Drama, Comedy, Action, etc.)  
- Search for movies  
- Dark mode and elegant UI design  
- Offline and connectivity handling  
- Clean, maintainable, and scalable architecture  

## Tech Stack

**Frontend:** Flutter (BLoC, Clean Architecture)  
**Backend:** Laravel (REST API, MySQL)  
**State Management:** BLoC  
**Database:** MySQL  
**Architecture:** Clean Architecture  
**Error Handling:** Advanced error and connectivity checks
